# Vue Pokedex

## Project setup
```
npm install
```

Open a new terminal and run

```
npm install json-server -g
cd /src/data
json-server pokemons.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
